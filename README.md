# 📚 Kirsh Manga Reader v1.0

Легкий, быстрый и стильный нативный ридер манги для Windows. Предназначен для комфортного чтения локальных коллекций манги в форматах папок и архивов, с автоматическим отслеживанием прогресса и гибкой настройкой интерфейса.

---

## ✨ Возможности

- 📂 **Поддержка форматов**: Чтение обычных папок с изображениями, а также архивов `.zip`, `.cbz`, `.rar` и `.cbr`.
- 📦 **Полная автономность**: Утилита `UnRAR.exe` уже встроена в установщик. Никаких дополнительных зависимостей или настроек путей не требуется.
- 🎨 **Кастомизация**: 7 темных цветовых схем и 6 акцентных цветов интерфейса на выбор.
- 🖼️ **Пользовательские обложки**: Возможность установить свою обложку для любой главы или папки (из файла на ПК или по прямой ссылке из интернета).
- 📊 **Отслеживание прогресса**: Автоматическое сохранение последней прочитанной страницы. Вкладки "Продолжить чтение" и "Завершено" для быстрого доступа.
- 🖥️ **Удобный ридер**: Плавный зум, подгонка по высоте окна, интуитивное управление мышью и клавиатурой.
- 🚀 **Профессиональная установка**: Готовый `.exe` установщик с ярлыками в меню "Пуск" и на рабочем столе, а также корректным удалением через систему.

---

## 📥 Установка

1. Скачайте файл установщика: `Kirsh_Manga_Reader_Setup.exe`.
2. Запустите его и следуйте инструкциям мастера установки.
3. По умолчанию программа установится в `C:\Program Files\Kirsh Manga Reader`.
4. Запустите программу через ярлык на рабочем столе или из меню "Пуск".
5. Откройте **Настройки** (⚙), укажите корневую папку с вашей коллекцией манги и нажмите "Сохранить изменения".

## 🎮 Управление

### В библиотеке:
| Действие | Клавиша / Мышь |
| :--- | :--- |
| Изменить размер карточек | Ползунок в верхней панели |
| Контекстное меню элемента | Правый клик по карточке |
| Перейти на уровень выше | Кнопка "← На уровень вверх" или Боковая кнопка мыши (Назад) |

### В читалке:
| Действие | Клавиша / Мышь |
| :--- | :--- |
| Следующая страница | `→` (Стрелка вправо) или Колесико мыши (вниз) |
| Предыдущая страница | `←` (Стрелка влево) или Колесико мыши (вверх) |
| Плавный зум (масштаб) | `Ctrl` + Колесико мыши |
| Сброс масштаба (по высоте) | Правый клик мыши (ПКМ) |
| Быстрый зум в точку | Двойной клик левой кнопкой мыши (ЛКМ) |
| Перейти к конкретной странице | Клик по тексту "X из Y" внизу окна |
| Полноэкранный режим | `F11` |
| Выход в библиотеку | `Esc` или Боковая кнопка мыши (Назад) |

## ⚙️ Технические детали

- **Язык**: Python 3.x
- **GUI фреймворк**: PyQt6
- **Сборка исполняемого файла**: PyInstaller (`--onefile`, `--noconsole`)
- **Создание установщика**: Inno Setup 6.x
- **Обработка RAR**: Встроенный `UnRAR.exe` (автоматически распаковывается во временную директорию при запуске)
- **Хранение настроек**: `%LOCALAPPDATA%\KirshMangaReader\` (гарантирует отсутствие ошибок прав доступа в `Program Files`)
