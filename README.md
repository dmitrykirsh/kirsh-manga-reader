# kirsh-manga-reader
Kirsh Manga Reader – читалка манги с ПК под Windows. Открывает .zip/.cbz/.rar/.cbr и папки с картинками. Запоминает последнюю страницу, показывает обложки, меняет тему и цвет акцента. Управление клавишами/мышью (зум, листание, полный экран). Удобная библиотека с вкладками «В процессе» и «Завершено». Легко менять обложки – из файла или по URL.

## Системные требования
- Windows 10 / 11
- Для работы с RAR требуется файл `UnRAR.exe` (положите рядом с программой или в папку с установщиком)

## Установка
1. Скачайте установщик из раздела [Releases](../../releases)
2. Запустите и следуйте инструкциям

## Первый запуск
1. Откройте настройки (кнопка ⚙)
2. Выберите корневую папку, где хранится ваша манга (папки с томами или архивы)
3. Наслаждайтесь чтением!

## Горячие клавиши (внутри читалки)
- `→` / `←` или колесо мыши – листание страниц
- `Ctrl` + колесо – масштаб
- `ПКМ` – сброс масштаба
- Двойной клик – зум в точку
- `Esc` – выход в библиотеку
- `F11` – полноэкранный режим
