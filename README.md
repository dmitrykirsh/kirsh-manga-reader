# 📚 Kirsh Manga Reader v2.0

Легкий, быстрый и стильный нативный ридер манги для Windows. Предназначен для комфортного чтения локальных коллекций манги в форматах папок и архивов, с автоматическим отслеживанием прогресса и **встроенным загрузчиком манги с сайтов**.

## ✨ Возможности

### 📥 Скачивание манги (новое в v2.0)
- **Поддержка сайтов**: MangaLib и Com-X.life
- **Авторизация через браузер**: достаточно войти 1 раз, куки и токены сохраняются
- **Выбор формата**: папки с картинками или CBZ (архив)
- **Автоматическая упаковка в CBZ**: тома для MangaLib, вся манга одним файлом для Com-X
- **Автообновление драйвера Chrome**: не нужно вручную скачивать chromedriver

### 📂 Чтение манги
- **Поддержка форматов**: обычные папки с изображениями, архивы `.zip`, `.cbz`, `.rar`, `.cbr`
- **Полная автономность**: утилита `UnRAR.exe` уже встроена в установщик
- **Оглавление**: быстрый переход к любой главе (работает с CBZ, папками, RAR/CBR)
- **Отслеживание прогресса**: автоматическое сохранение последней прочитанной страницы
- **Вкладки**: "Библиотека", "Продолжить чтение", "Завершено"

### 🎨 Интерфейс и управление
- **Контекстное меню**: удалить прогресс, отметка о прочтении, смена обложки, удаление с устройства
- **Открыть в проводнике**: выделить файл/папку из контекстного меню
- **Пользовательские обложки**: установка из файла на ПК или по прямой ссылке
- **Удобный ридер**: плавный зум, подгонка по высоте окна, интуитивное управление мышью и клавиатурой

### 🚀 Установка
- Готовый `.exe` установщик с ярлыками в меню "Пуск" и на рабочем столе
- Корректное удаление через систему

## 📥 Установка

1. Скачайте файл установщика: `kirsh.manga.reader.install.exe`
2. Запустите его и следуйте инструкциям мастера установки
3. По умолчанию программа установится в `C:\Program Files\Kirsh Manga Reader`
4. Запустите программу через ярлык на рабочем столе или из меню "Пуск"
5. Откройте **Настройки** (⚙), укажите корневую папку с вашей коллекцией манги и нажмите "Сохранить изменения"

## 🎮 Управление

### В библиотеке:
| Действие | Клавиша / Мышь |
| :--- | :--- |
| Контекстное меню элемента | Правый клик по карточке |
| Перейти на уровень выше | Кнопка "← На уровень вверх" или Боковая кнопка мыши (Назад) |

### В читалке:
| Действие | Клавиша / Мышь |
| :--- | :--- |
| Следующая страница | `→` (Стрелка вправо) или Колесико мыши (вниз) |
| Предыдущая страница | `←` (Стрелка влево) или Колесико мыши (вверх) |
| Плавный зум (масштаб) | `Ctrl` + Колесико мыши |
| Сброс масштаба (по высоте) | Правый клик мыши (ПКМ) |
| Быстрый зум в точку | Двойной клик левой кнопкой мыши (ЛКМ) |
| Перейти к конкретной странице | Клик по тексту "X из Y" внизу окна |
| Оглавление | Кнопка "📑 Оглавление" вверху окна |
| Полноэкранный режим | `F11` |
| Выход в библиотеку | `Esc` или Боковая кнопка мыши (Назад) |

## 🔑 Авторизация для скачивания

Для скачивания манги с сайтов可能需要 авторизация:

| Сайт | Что нужно |
| :--- | :--- |
| **MangaLib** | Токен (для манги 18+). Получается автоматически через браузер при нажатии "Получить токен" |
| **Com-X.life** | Куки. Получаются автоматически через браузер при нажатии "Авторизоваться" |

**Важно**: для авторизации требуется установленный браузер Google Chrome.

## ⚙️ Хранение данных

- **Настройки программы**: `%LOCALAPPDATA%\KirshMangaReader\reader_config.json`
- **Токен MangaLib**: `%LOCALAPPDATA%\KirshMangaReader\mangalib_token.json`
- **Куки Com-X.life**: `%LOCALAPPDATA%\KirshMangaReader\comx_cookies.json`
- **Пользовательские обложки**: `%LOCALAPPDATA%\KirshMangaReader\custom_covers\`
